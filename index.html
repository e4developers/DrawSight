What DrawSight is built to help with

The app is focused on solo inspectors who want to stay fast in the field without giving up control over client-facing reports.

  1. Capture photos, notes, and findings directly on-site.
  2. Use AI to draft language and classify photo-heavy batches faster.
  3. Review, edit, and approve everything before export.
  4. Generate reports and backups without depending on a full always-on cloud stack.

Built around the real inspection flow

DrawSight is not trying to be a generic form app. It is built around job-site capture, review speed, report clarity, and keeping the inspector in charge of the final result.

Current Strengths

Where the app is strongest right now

  • mobile-first usage for solo inspectors

Release Reality

What the current build does today

  • stores inspection data primarily on-device
  • sends AI requests directly to Anthropic when enabled
  • uses a user-supplied Anthropic API key in current builds
  • lets inspectors choose when reports and files leave the app
