Privacy, terms, and support for DrawSight.

DrawSight is a local-first mobile inspection app built for fast field reporting, photo-heavy workflows, and optional AI-assisted drafting. These pages explain how the app currently handles data and what users agree to when they use it.

Current release snapshot

  • Inspection data is primarily stored on-device.
  • AI requests currently go directly to Anthropic when enabled.
  • Current builds still use a user-supplied Anthropic API key.
  • Exports and backups leave the app only when the inspector chooses to share them.

Local-first by design

DrawSight does not currently run a first-party always-on sync platform for inspectors. Reports, photos, notes, and recovery data are primarily managed on the device.

Optional third-party processing

When AI tools are used, relevant photos, prompts, and notes may be sent to Anthropic. When address autocomplete is used, typed property queries may be sent to the configured lookup provider.

Inspector-controlled exports

Reports, backups, and logs are exported only through user-initiated share or file-picker flows. DrawSight does not silently push those files to a DrawSight server in current builds.
